>Commands
To switch to another user, run:

    su - username

>To run command if you do not have root privileges.

    sudo [command to run]

>Edits the sudo configuration file (`/etc/sudoers`) to add or remove rights for users to run certain commands

    visudo

>Location in `/etc/sudoers` to add the group to run sudo commands
