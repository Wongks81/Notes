<h2> Tenants, Subscriptions and Licensing </h2>

Subscription:
- An Agreement with Microsoft to use one or more Microsoft cloud platforms or services
- Charges accrue based on either a <b>per-user license fee</b> or on <b>cloud based resource consumption</b>

Different types of licenses:
- Microsoft SaaS cloud offerings
  - A license allows a specific user account to use the services of the cloud offering
  - Charge by monthly fee
  
- Azure PaaS based cloud services
  - Software licenses are built in the service pricing
  
- Azure IaaS based virtual machines
  - Additional licenses use the software or application installed on a virtual machine image might be required
  - Some virtual machine images have licensed versions of software installed and cost are charged by <b> per minute</b> rate for the server.

User accounts
- Stored in Azure Active Directory (AAD) tenant

Azure AD tenant:
- Is a specific instance of Azure AD containing accounts and groups
  
- Can be synchronized with exiisiting Active Directory Domain Services (AD DS) accounts using <b>Azure AD connect</b>
