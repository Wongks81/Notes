<h2> Containers </h2>

<h4>Docker Containers</h4>
- Standarized packaging for software and dependencies

- A way to isolate apps from each other

- Works with Linux and Windows

- Allows separate apps to share the same OS kernel

Docker Basics

| Name          | Description                                                                                                                               |
| ------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| Image         | The main component / basis of a Docker container <br> Content is at rest.                                                                 |
| Container     | Image when it is "ruinning" <br> Standard unit for app service                                                                            |
| Engine        | The software that executes commands for containers <br><br> Networking and volumes are part of the engine. <br> Can be clustered together |
| Registry      | Stores , distributes and manages Docker containers                                                                                        |
| Control Plane | Management plane for container and cluster orchestration                                                                                  |
