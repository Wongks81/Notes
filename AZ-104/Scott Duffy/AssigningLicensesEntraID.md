# Assigning Licenses to Users in Entra ID

1. Go to Users Page in Azure

    ![](images/2026-05-18-06-38-55.png)

2. Click on the User that you want to assign license to

3. Click on the license button on the submenu on the left

    ![](images/2026-05-18-06-40-23.png)

4. Click on the "+ Assignment" on top menu

    ![](images/2026-05-18-06-41-03.png)

5. The licenses that is currently available will be shown in the next page

6. Assign what is needed

    ![](images/2026-05-18-06-42-32.png)

7. Click on "Save"
---

### Troubleshooting

* Due to Geological locations laws and restrictions, you might encounter errors when assigning.
  
* One of the reasons might be that the Usage location for the user is not defined in the user properties

    ![](images/2026-05-18-06-45-40.png)
